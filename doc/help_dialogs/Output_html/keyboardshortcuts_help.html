KEYBOARD SHORTCUTS
Keys Description
ENTER Turns ON/OFF Keyboard Shortcuts
SPACE When Keyboard Shortcuts are ON chooses the current button
When Keyboard Shortcuts are OFF adds a custom event
LEFT,RIGHT,UP,DOWN Move background or key focus
a Autosave
CRTL+N Autosave + Reset + START
t   [Windows: CTRL+SHIFT+t] Toggle mouse cross lines
d Toggle xy scale (T/Delta)
c Shows/Hides Controls
x Shows/Hides LCD Readings
m Shows/Hides Event Buttons
b Shows/Hides Extra Event Buttons
s Shows/Hides Event Sliders
p Toggle PID mode
h   [Windows: CTRL+SHIFT+h] Load background profile
ALT+h   [Windows: CTRL+ALT+h] Remove background profile
l Load alarms
+,- Inc/dec PID lookahead
CRTL 0-9 Changes Event Button Palettes
; Application ScreenShot
: Desktop ScreenShot
q,w,e,r + <i>nn</i> Quick Custom Event
v + <i>nnn</i> Quick PID SV
f   [Windows: CTRL+SHIFT+f] Full Screen Mode

MODBUS
The MODBUS device corresponds to input channels1 and 2.. The MODBUS_34 extra device adds input channels 3 and 4.

Inputs with slave id set to 0 are turned off.

Modbus function 3 "read holding register" is the standard. Modbus function 4 triggers the use of "read input register". Input registers (fct 4) usually are from 30000-39999.

Most devices hold data in 2 byte integer registers. A temperature of 145.2C is often sent as 1452. In that case you have to set the divider to "x/10".

Few devices hold data as 4 byte floats in two registers. Tick the Float flag in this case.
